PROJECT OVERVIEW:-
> The Weather App is a simple and responsive web application that allows users to search for any city and view real-time weather information. It uses the free API from Open-Meteo to fetch live weather data without requiring an API key.

> The app displays important weather details such as temperature, wind speed, humidity, visibility, and the current date. Users can search for different cities, and the interface updates dynamically with the latest data.

> It also includes a Dark/Light mode toggle for better user experience and is fully responsive, meaning it works smoothly on both desktop       and mobile devices.
